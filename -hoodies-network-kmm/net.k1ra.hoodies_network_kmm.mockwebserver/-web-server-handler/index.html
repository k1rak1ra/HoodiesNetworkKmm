open class WebServerHandler

Base class for handling requests in the MockWebServer For every endpoint being served by the MockWebServer, a class inheriting from WebServerHandler must be made Then, the class needs to override handleRequest(call: HttpCall) in order to handle the request using a ktor-like syntax
